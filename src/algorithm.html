Algorithms

List of algorithms you can learn about

Bubble sort

Good, Sorting

A simple sorting algorithm that repeatedly steps through the list, compares adjacent elements and swaps them if they are in the wrong order.

Quick sort

Great, Sorting

A efficient sorting algorithm commonly used algorithm for sorting.It is known to be the most efficent due to its perfect time complexity.

Merge sort

Great, Sorting

A efficent and stable divide and conquer algorithm that is comparison-based and is similar to stable sort. It was invented by John von Neumann in 1945

Insertioin Sort

Bad, Sorting

A simple sorting algorithm that builds the final sorted array one item at a time.However, it works well when having a list thats close to sorted

Heap sort algoritm

Good, Sorting

Heapsort is a efficent and stable comparison-based sorting algorithm. Its main purpose is transforming the list of items to be sorted into a heap data structure, or a binary tree

Linear Search Algorithm

Good, Sorting

A linear search is a efficent sorting algorithm for finding an element within a list.It checks each element of the list until a match is found or the whole list has been searched

Radix sort algorithm

Bad, Non-comparison sort

A non-comparative stable sorting algorithm. It avoids comparison by creating and distributing elements into buckets according to their radix and is espeically used in arrays

Minimax Algoritm

Great, Recursive, Game

A recursive or backtracking algorithm which is used in decision-making and game theory. Games such as chess or tic tac toe use minimax as a 2nd player for a user. Its also pretty simple to implement

Binary Search algoritm

Good, Sorting, Search

Binary search, also known as half-interval search, logarithmic search, or binary chop, is a search algorithm that finds the position of a target value within a sorted array

Gnome sort algorithm

Sorting, Ok

Gnome sort or Stupid sort is a sorting algorithh that takes each item at a time and gets the item to the proper place by a series of swaps. Its also similar to Insertion Sort.

Introsort algorithm

Selectioin, Bad

Introsort or introspective sort is a hybrid sorting algorithm that provides both fast average performance and optimal worst-case performance

Flash Sort algorithm

Distribution, Ok

Flashsort is a distribution sorting algorithm proceeds by finding the smallest (or largest, depending on sorting order) component in the unsorted sublist
